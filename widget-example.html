🏠 Nexo Inmobilia Widget - Demo

Esta página demuestra cómo integrar y usar el widget de chat de Nexo Inmobilia.

📋 Integración Básica

Solo necesitas incluir el script JavaScript (el CSS viene incluido):

<!-- Widget Container --> <div id="nexo-widget"></div> <!-- Solo un archivo JS necesario --> <script src="https://cdn.tudominio.com/nexo-widget.iife.js"></script>
